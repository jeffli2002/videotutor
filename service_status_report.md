可以使用以下命令测试各个API服务：

```bash

  -H "Content-Type: application/json" \
  -d '{"model":"qwen-turbo","messages":[{"role":"user","content":"测试"}]}'

# 测试Manim API

  -H "Content-Type: application/json" \
  -d '{"script":"print(\"Hello World\")","type":"test"}'

curl -X POST http://localhost:8003/api/tts \
  -H "Content-Type: application/json" \

```
